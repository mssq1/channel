YouTube Multi-Video Player

Play 10 random videos from any YouTube channel simultaneously

Get a free API key from Google Cloud Console. Enable the YouTube Data API v3.
Enter your YouTube API key and a channel URL above, then click "Load Random Videos"

How to use:

1. Get a YouTube Data API v3 key from Google Cloud Console

2. Enable the YouTube Data API v3 for your project

3. Enter your API key and a YouTube channel URL (or channel ID)

4. Click "Load Random Videos" to fetch videos from that specific channel

Note: The YouTube API has quota limits. Each request consumes quota points.
